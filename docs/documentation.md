# OpenTrack Documentation

## Project Overview
OpenTrack is a Python-based program that manages an open source project by collecting and processing user input data.

## Features
- Collect contributor information from user input
- Track and manage issues
- Analyze issue priority and status
- Identify top reporter and urgent issues
- Save results into text and CSV files

## Data Structures Used
- Tuple: used for fixed project information
- List: used to store contributors and issues
- Dictionary: used for structured data and counting
- Set: used for unique values and analysis

## How it works
The program asks the user to input contributor and issue information.
It processes the data using Python data structures such as lists, dictionaries, and sets.

It then performs analysis like counting priorities and finding the most active reporter.
Finally, it saves the results into report files.

## Output Files
- project_report.txt
- issues.csv
